# 🌱 Predictive Analysis of Renewable Energy Adoption Trends

## 📌 Overview
This project focuses on predicting renewable energy adoption trends using Machine Learning and Deep Learning techniques. The goal is to analyze historical energy data and forecast future trends to support sustainable energy planning.

---

## 🚀 Features
- 📊 Data preprocessing and visualization
- 🤖 Multiple ML models implemented
- 🔥 Deep learning using LSTM
- 📈 Time-series forecasting (ARIMA)
- 📉 Model comparison and evaluation

---

## 🧠 Models Used
- Linear Regression
- Random Forest
- XGBoost
- AdaBoost
- ARIMA
- LSTM (Best Performing Model)

---

## 📊 Results
- LSTM achieved highest accuracy
- XGBoost performed best among ML models
- Renewable energy shows increasing trend globally
